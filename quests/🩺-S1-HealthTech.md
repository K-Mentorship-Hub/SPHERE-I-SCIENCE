## 📖 Theory Bite

HealthTech and biomedical work sits at the intersection of **evidence, regulation, and human impact**.

| Concept | What it means |
|---------|---------------|
| Evidence hierarchy | Systematic reviews > RCTs > Cohort > Case reports > Expert opinion |
| Regulatory reality | FDA/EMA approval, clinical trials, post-market surveillance |
| Translation gap | 10-17 years from lab discovery to clinical use |
| Ethics | Informed consent, HIPAA/GDPR, equity in access |

**Daily reality:** Literature review, protocol writing, data analysis, navigating bureaucracy. The payoff: your work can literally save lives.

## ⚔️ Quests

### Quest S1.1 — Glossary Builder
**⏱ Time:** ~45 min | **Difficulty:** ⭐ Beginner

**Goal:** Build a plain-language glossary that makes a biomedical sub-area accessible to newcomers.

**Steps:**
1. Pick one sub-area (e.g., wearable diagnostics, telemedicine, genomics, RNA therapeutics)
2. Identify 10 key terms that newcomers always stumble on
3. For each term, write:
   - **Definition** (1 sentence, no jargon)
   - **Why it matters** (1 sentence, real-world context)

<details>
<summary>💡 Stuck? Click for example entries</summary>

| Term | Definition | Why it matters |
|------|-----------|----------------|
| Biomarker | A measurable signal that indicates a biological state or condition | Used for early disease detection and monitoring treatment response |
| In vitro | Experiments done outside a living organism (e.g., in a test tube) | Cheaper and faster than clinical trials, but results don't always translate to humans |
| Off-target effect | When a drug or therapy affects something other than its intended target | The main reason "miracle drugs" fail in clinical trials |

</details>

**Reflection:** Did you enjoy translating complexity into clarity?

### Quest S1.3 — Problem Map
**⏱ Time:** ~60 min | **Difficulty:** ⭐⭐ Intermediate

**Goal:** Map a health problem into a structured opportunity for technology intervention.

**Steps:**
1. Pick one health problem you care about
2. Fill in the map:

```
PROBLEM MAP
├── 🎯 What is the health problem?
├── 👥 Who is affected? (be specific — age, geography, severity)
├── 💊 What current solutions exist?
├── ❌ Where do they fail?
├── 🔧 What could technology do differently?
└── 📏 One metric for success
```

<details>
<summary>💡 Need a problem to start with?</summary>

Try one of these:
- Post-surgical infection detection is delayed by 24-48 hours in low-resource hospitals
- Mental health screening tools miss 40% of depression cases in adolescents
- Rare disease patients wait an average of 7 years for correct diagnosis
- Medication adherence drops below 50% for chronic conditions after 6 months

</details>

**Reflection:** Did you naturally start thinking about solutions, or did you want to understand the problem deeper first?

### Quest S1.4 — Evidence Rating
**⏱ Time:** ~45 min | **Difficulty:** ⭐⭐ Intermediate

**Goal:** Rate the evidence behind health claims you see in the wild.

**Steps:**
1. Find 3 health claims on social media, news, or product packaging
2. For each claim, trace it to its source
3. Rate using the evidence hierarchy:

| Level | Type | Your claim's level |
|-------|------|--------------------|
| 5 | Systematic review / meta-analysis | |
| 4 | Randomized controlled trial | |
| 3 | Cohort study | |
| 2 | Case report / case series | |
| 1 | Expert opinion / anecdote | |
| 0 | No evidence found | |

4. How many of the 3 claims hold up?

**Reflection:** Did this make you more skeptical? More curious? Both?

### Quest S1.5 — Regulatory Obstacle Course
**⏱ Time:** ~60 min | **Difficulty:** ⭐⭐⭐ Advanced

**Goal:** Map the regulatory path for a hypothetical health product.

**Steps:**
1. Pick a hypothetical product (e.g., AI skin-cancer screening app, wearable glucose monitor, at-home genetic test)
2. Research the regulatory steps in your region (FDA, EMA, or local equivalent)
3. Create a timeline:

| Phase | What happens | Duration | Cost estimate |
|-------|-------------|----------|---------------|
| Pre-clinical | | | |
| Phase I trial | | | |
| Phase II trial | | | |
| Phase III trial | | | |
| Regulatory review | | | |
| Post-market surveillance | | | |

<details>
<summary>💡 Quick reference for FDA pathway</summary>

- **510(k)**: For devices similar to existing ones — ~3-6 months, lower cost
- **De Novo**: For novel low-to-moderate risk devices — ~1 year
- **PMA (Pre-market Approval)**: For high-risk devices — ~1-3 years, expensive
- **Digital Health**: FDA has a specific pathway for software as medical device (SaMD)

</details>
