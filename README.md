## What Is SPHERE I?

`SPHERE I / SCIENCE` is the domain-facing layer inside `K Mentorship Hub`.

Use this sphere when the main question is:

`What are we trying to understand, investigate, test, measure, or explain?`

This sphere helps people:

- find science-first paths without getting buried in jargon
- connect research-facing thinking to practical skills
- translate domain problems into project, MVP, or contribution ideas
- move from curiosity to a structured direction

## Practical Rule

Do not force every science-facing idea into a pure science box forever.

Instead:

- choose a `primary_sphere`
- add `secondary_spheres` if needed
- add a `combo` like `S+T` or `S+E`

That keeps structure clean while preserving hybrid logic.
