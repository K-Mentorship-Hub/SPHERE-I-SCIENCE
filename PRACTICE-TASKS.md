# Practice Tasks · SPHERE I / SCIENCE

## Starter Tasks

### `S1` · HealthTech / Bio / Med

1. Create a plain-language glossary for 10 key terms in one chosen sub-area.
2. Compare three public resources and explain which one is best for a beginner.
3. Draft a one-page `problem map` for a science-informed MVP idea.

### `S2` · Green / Sustainability

1. Map one climate or sustainability problem into `problem -> actors -> possible interventions`.
2. Build a simple resource list with `what to read first / second / later`.
3. Write a mini note: `Where does this become S, E, T, or S+E+T?`

### `S3` · AgTech & Food Security

1. Create a short comparison of `AgriTech`, `FoodTech`, and `Agri-data systems`.
2. Draft one simple `starter project` brief for a learner.
3. Make a visual checklist for how to evaluate an agri problem.

### `S4` · Human Systems / Learning

1. Make a learning experiment template for a 7-day self-tracking challenge.
2. Summarize one research-facing topic into a beginner note.
3. Draft a `how to learn this without chaos` guide.
